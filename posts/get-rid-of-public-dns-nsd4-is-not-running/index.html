It’s really annoying to see cross marks on status checks on Mail-in-a-Box

Solution:

sudo sysctl -w net.ipv6.conf.all.disable_ipv6=0

sudo sysctl -w net.ipv6.conf.default.disable_ipv6=0

sudo service nsd start
